## Error Codes

The following table lists the possible error codes you may encounter when using the API:

| HTTP Code | Error Code | Message                        | Description                                        |
| --------- | ---------- | ------------------------------ | -------------------------------------------------- |
| 400       | 1001       | Invalid request parameters     | Request parameters are invalid or missing          |
| 400       | 1002       | Invalid model name             | The specified model name is not supported          |
| 400       | 1003       | Prompt too long                | The prompt exceeds the maximum allowed length      |
| 401       | 2001       | Invalid API key                | The provided API key is invalid or expired         |
| 401       | 2002       | Missing authorization header   | Authorization header is missing from the request   |
| 403       | 3001       | Insufficient quota             | Your account has insufficient quota for this operation |
| 404       | 4001       | Task not found                 | The specified task ID does not exist               |
| 429       | 5001       | Rate limit exceeded            | Too many requests in a short period of time        |
| 500       | 6001       | Internal server error          | An unexpected error occurred on the server         |
| 503       | 6002       | Service unavailable            | The service is temporarily unavailable             |
